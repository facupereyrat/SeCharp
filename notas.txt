int: Es una abreviatura de "integer" (entero). Se utiliza para almacenar números enteros, es decir, números sin decimales. 
Por ejemplo, 5, -3 y 42 son valores de tipo int.

string: Es una cadena de caracteres. Se utiliza para almacenar texto. 
Por ejemplo, "Hola", "1234" y "Programación" son valores de tipo string.

double es un tipo de dato que se utiliza para almacenar números con decimales. 
Es una abreviatura de "double-precision floating-point number" (número de punto flotante de doble precisión).

try y catch permiten que el programa continúe ejecutándose de manera controlada.
Si ocurre una excepción dentro del bloque try, el flujo del programa se transfiere al bloque catch correspondiente.
catch: Este bloque contiene el código que se ejecuta cuando se produce una excepción en el bloque try.
