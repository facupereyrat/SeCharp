Los operadores aritméticos en C# son fundamentales para realizar cálculos matemáticos. 
Aquí tienes una lista de los operadores aritméticos más comunes:

Suma (+): Añade dos operandos.

int a = 5;
int b = 3;
int resultado = a + b; // resultado es 8

Resta (-): Resta el segundo operando del primero.

int a = 5;
int b = 3;
int resultado = a - b; // resultado es 2

Multiplicación (*): Multiplica dos operandos.

int a = 5;
int b = 3;
int resultado = a * b; // resultado es 15

División (/): Divide el primer operando por el segundo. Ten en cuenta que si ambos operandos son enteros, el resultado será un entero (división entera).

int a = 5;
int b = 3;
int resultado = a / b; // resultado es 1 (división entera)

Módulo (%): Devuelve el resto de la división del primer operando por el segundo.

int a = 5;
int b = 3;
int resultado = a % b; // resultado es 2
Estos operadores son esenciales para cualquier tipo de cálculo en C#
